# France Launcher - Site Officiel

Bienvenue sur le dépôt du site officiel de **France Launcher**, le launcher Minecraft optimisé pour le PVP et la performance.

## 🚀 Fonctionnalités du Site

- **Design Moderne & Gaming** : Interface sombre avec dégradés dynamiques aux couleurs de la France (Bleu, Blanc, Rouge).
- **Multi-Pages** : 
  - Accueil (Hero, Features, Communauté)
  - Page de Téléchargement dédiée
  - Pages Légales (License, CGU)
  - Page de Contact avec formulaire
- **Optimisé Mobile** : Entièrement responsive avec menu mobile.
- **Technologies** : React, Vite, Tailwind CSS, Lucide React, React Router.

## 🛠️ Installation Locale

1. Clonez le dépôt :
   ```bash
   git clone https://github.com/VOTRE_PSEUDO/france-launcher-site.git
   ```

2. Installez les dépendances :
   ```bash
   npm install
   ```

3. Lancez le serveur de développement :
   ```bash
   npm run dev
   ```
